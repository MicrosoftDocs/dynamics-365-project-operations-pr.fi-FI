<span data-ttu-id="338a2-105">Kun resurssipäällikkö ehdottaa nimettyä resurssia täyttämään projektin yleisen resurssipyynnön, yleisen ryhmänjäsenen **Pyynnön tila** -kenttä päivittyy arvoon **Tarkistettava**.</span><span class="sxs-lookup"><span data-stu-id="338a2-105">When a resource manager proposes a named resource to fill the generic resource request for a project, the **Request Status** field for the generic team member will be updated to **Needs Review**.</span></span> <span data-ttu-id="338a2-106">Pyyntö lähetetään projektipäällikölle hyväksyttäväksi tai hylättäväksi.</span><span class="sxs-lookup"><span data-stu-id="338a2-106">The request will be sent to the project manager for approval or rejection.</span></span>

![Yleinen ryhmän jäsen ja ehdotus](media/RM-how-to-19.png)

<span data-ttu-id="338a2-108">**Projektiryhmän jäsen** -sivun **Ehdotetut resurssit** -välilehden ruudukossa näkyvät ehdotetun resurssin kulloisetkin varaukset.</span><span class="sxs-lookup"><span data-stu-id="338a2-108">The grid on the **Proposed Resources** tab on the **Project Team Member** page shows the proposed resource’s current bookings.</span></span> <span data-ttu-id="338a2-109">Kun ehdotus on hyväksytty, ruudukko päivitetään vastaamaan kyseistä varausta.</span><span class="sxs-lookup"><span data-stu-id="338a2-109">After the proposal is accepted, the grid is updated to reflect that booking.</span></span>

![Resurssiehdotuksen hyväksyminen](media/RM-how-to-20.png) 

<span data-ttu-id="338a2-113">Samoin kuin yleisen resurssipyynnön täyttämisessä suoraan nimetyllä resurssilla, yleinen resurssi korvataan ja kohdennetut tehtävät päivitetään nimetyllä ryhmän jäsenellä.</span><span class="sxs-lookup"><span data-stu-id="338a2-113">Similar to directly fulfilling a generic resource request with a named resource, the generic resource will be replaced and the assigned tasks will be updated with the named team member.</span></span>
