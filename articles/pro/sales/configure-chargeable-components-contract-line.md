# <a name="configure-chargeable-components-of-a-project-based-contract-line---lite"></a><span data-ttu-id="14a0a-103">Projektipohjaisen sopimusrivin veloitettavien komponenttien määrittäminen – lite</span><span class="sxs-lookup"><span data-stu-id="14a0a-103">Configure chargeable components of a project-based contract line - lite</span></span>

<span data-ttu-id="14a0a-105">Projektipohjaiseen sopimusriviin on *sisällytetty* komponentteja ja *laskutettavia* osia.</span><span class="sxs-lookup"><span data-stu-id="14a0a-105">A project-based contract line has *included* components and *chargeable* components.</span></span>

<span data-ttu-id="14a0a-106">Sisältyvät komponentit ovat komponentteja, joihin liittyy:</span><span class="sxs-lookup"><span data-stu-id="14a0a-106">Included components are components that are subject to:</span></span>

  - <span data-ttu-id="14a0a-113">Ei veloitettava</span><span class="sxs-lookup"><span data-stu-id="14a0a-113">Non-chargeable</span></span>

<span data-ttu-id="14a0a-114">Laskutettavat komponentit voidaan määrittää tehtäville, rooleille ja tapahtumaluokille.</span><span class="sxs-lookup"><span data-stu-id="14a0a-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

### <a name="update-a-project-task-as-chargeable-or-non-chargeable"></a><span data-ttu-id="14a0a-121">Projektitehtävän päivittäminen laskutettavana tai ei-laskutettavana</span><span class="sxs-lookup"><span data-stu-id="14a0a-121">Update a project task as chargeable or non-chargeable</span></span>
