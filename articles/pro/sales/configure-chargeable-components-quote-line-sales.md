<span data-ttu-id="9a3ce-105">Projektipohjaiseen tarjousrivillä on *sisällytettyjen* osien ja *laskutettavien* osien käsitteet.</span><span class="sxs-lookup"><span data-stu-id="9a3ce-105">A project-based quote line has the concept of *included* components and *chargeable* components.</span></span>

<span data-ttu-id="9a3ce-106">Sisällytettyjä osia voivat olla seuraavat:</span><span class="sxs-lookup"><span data-stu-id="9a3ce-106">Included components are subject to:</span></span>

  - <span data-ttu-id="9a3ce-113">Ei veloitettava</span><span class="sxs-lookup"><span data-stu-id="9a3ce-113">Non-chargeable</span></span>

<span data-ttu-id="9a3ce-114">Laskutettavat komponentit voidaan määrittää tehtäville, rooleille ja tapahtumaluokille.</span><span class="sxs-lookup"><span data-stu-id="9a3ce-114">Chargeable components can be defined on tasks, roles, and transaction categories.</span></span>

<span data-ttu-id="9a3ce-115">Maksukyky määritetään tarjousrivin tehtäville, ja se koskee kaikkia siihen riviin sisältyviä tapahtumaluokkia.</span><span class="sxs-lookup"><span data-stu-id="9a3ce-115">Chargeability is defined on the tasks for a quote line and applies to all transaction classes included on that line.</span></span> <span data-ttu-id="9a3ce-116">Jos sopimusrivin **Sisällytä tehtävät** -kentän asetus on **Koko projekti** tai se on jätetty tyhjäksi, **Veloitettavat tehtävät** -välilehti ei ole käytettävissä.</span><span class="sxs-lookup"><span data-stu-id="9a3ce-116">If the **Include Tasks** field is set to **Entire project** or left blank, the **Chargeable Tasks** tab isn't available.</span></span>

<span data-ttu-id="9a3ce-122">Projektitehtävä voi olla laskutettava tai ei-laskutettava tietyn projektipohjaisen tarjousrivin yhteydessä, mikä mahdollistaa seuraavan asennuksen:</span><span class="sxs-lookup"><span data-stu-id="9a3ce-122">A project task can be chargeable or non-chargeable in the context of a specific project-based quote line, which makes the following setup possible:</span></span>
